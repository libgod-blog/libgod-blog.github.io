一款精致的VIM配置vim_config发布

操作系统
  • Vim

vim_config一款非常精致的vim配置，集成常用的插件，精美的配色，使用方法简单。如果你是一个vim新手，vim_config正适合你。如果你是vim高手，但又讨厌复杂的vim配置，vim_config同样适合你。

内容更新：

  1. 简化安装步骤。把项目pull下来，只要运行安装文件install.sh就好了。
  2. 精美的配色。采用流行的molokai配色方案，如果你经常写代码，这款配色再适合不过了。
  3. 高效的多文件编辑。能同时编辑多个文件，轻易的在多个文件之间切换。
  4. 轻松打开文件。只要按快捷键Ctrl+p，输入文件名，就能智能匹配要打开的文件。
  5. 方便的快捷键。有一些常用的快捷键，操作轻松自如。
  6. 自动添加文件头。新建一些常用格式的文件时，会智能插入文件头，例如签名、创建时间、修改时间等。

除了列举的几项，还有很多隐藏BUFF等待着你，速速前往下载。
